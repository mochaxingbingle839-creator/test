# 计算机图形学大作业

本项目保留 Python 桌面增强版矢量图形编辑器。

## 运行方式

目录：`python_version`

```bash
cd python_version
pip install -r requirements.txt
python vector_editor.py
```

主要文件：

- `vector_editor.py`：Python/Tkinter 增强版矢量图形编译器
- `PYTHON_README.md`：Python 版说明
- `requirements.txt`：Pillow 与 NumPy 依赖

Python 增强版新增 DSL 编译器、SVG 导出、高清 PNG 导出、图形统计分析、撤销/重做、图层列表、网格吸附和更多图元。

## 其他文件

- `计算机图形学上机大作业-2026.pdf`：作业要求
- `111.json`、`name.json`：已保存的图形数据文件，可用 Python 版加载
